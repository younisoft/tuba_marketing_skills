# Copy Scorecard

> **Part of:** [AI_KNOWLEDGE_PLATFORM.md](../AI_KNOWLEDGE_PLATFORM.md)
> **Purpose:** the numeric, 1-10 scored counterpart to [content-review-checklist.md](../advertising-system/content-review-checklist.md)'s pass/fail checklist. Use the checklist to catch binary violations fast; use this scorecard when you need a graded, comparative read (e.g., choosing between two headline drafts, or tracking copy quality over time).
> **Owner:** Copy lead
> **Review frequency:** semi-annually

---

## Scoring Dimensions (1-10 each)

### 1. Headline
Formula-driven (per [headline-library.md](../brand-intelligence/headline-library.md)), 5-8 words, survives standalone (visual-system.md §9.7's OOH test).

### 2. Hook
Names a real human moment before mentioning Tuba/the product; specific, not abstract (prompt-patterns.md §5).

### 3. CTA
Exactly one, specific to the literal next action, drawn from [cta-library.md](../brand-intelligence/cta-library.md).

### 4. Persuasion
Objection pre-empted (Reassure stage), specificity over abstraction, no forbidden technique used outside its guardrail (psychological-triggers.md).

### 5. Readability
Passes a "read aloud" test — sounds like a person talking. Appropriate length for format.

### 6. Grammar
No spelling/grammatical errors in either language; consistent editorial style (copywriting.md §7.3).

### 7. Saudi Localization
Natively-written Arabic (never translated), correct register (copywriting.md §7.1), Western numerals, no imported-phrasing tells (prompt-patterns.md §4).

### 8. Brand Tone
Reflects all five voice adjectives (Warm/Clear/Grounded/Guiding/Respectful); customer is the grammatical hero.

### 9. Clarity
One idea per sentence; any regulatory term plain-language-defined on first use (copywriting.md §5.4).

### 10. Luxury Register (score only if applicable)
For luxury-segment copy: restraint and specificity, zero label-adjectives ("luxurious," "exclusive"). N/A for standard-register copy.

## Quick-Fail Override

Regardless of numeric average, an automatic REJECT applies if the copy contains: a forbidden expression (copywriting.md §8.3), an unverifiable superlative, fabricated urgency/scarcity/social proof, more than one CTA, or literal-translation Arabic. These mirror [content-review-checklist.md](../advertising-system/content-review-checklist.md)'s Quick Fail Conditions exactly — run that checklist first if you suspect any of these, since it's faster than full scoring.

## Best Practices
- Use this scorecard comparatively (score Draft A vs. Draft B) more often than as a pass/fail gate alone — the checklist already handles binary gating
- Track scores over time in [../brand-memory/headline-performance.md](../brand-memory/headline-performance.md) once real performance data exists, to see whether high scorecard scores actually correlate with real results

## Common Mistakes
- Running this scorecard instead of the faster checklist for routine pre-publish gating — use the checklist for that, this scorecard for graded/comparative decisions
- Scoring Luxury Register on standard-register copy — mark N/A, don't penalize copy for correctly not using restrained luxury language
