# Marketing Insights — Tuba

> **Layer:** Brand Memory (living)
> **Purpose:** insights derived from Tuba's *own* marketing activity — patterns noticed across campaigns, audience behavior, and content performance. Distinct from [market-observations.md](market-observations.md) (external market/industry trends unrelated to a specific Tuba campaign) and [customer-feedback.md](customer-feedback.md) (direct customer language).
> **Owner:** Marketing Strategy lead
> **Update trigger:** as insights emerge from campaign analysis; reviewed monthly
> **Related documents:** [campaign-history.md](campaign-history.md), [monthly-retrospectives.md](monthly-retrospectives.md), [market-psychology.md](../brand-intelligence/market-psychology.md)

## Entry Template

```markdown
## [Insight title] — [date]

**Observation:** [what pattern was noticed, across which campaigns/data]
**Evidence:** [links to campaign-history.md entries, ab-tests.md results, or other data supporting this]
**Implication:** [what this means for future strategy or execution]
**Confidence:** [emerging/confirmed — has this been seen once or repeatedly?]
**Action:** [what should change as a result, if anything]
**Logged by:** [name/role] on [date]
```

## Best Practices
- Always link an insight to real evidence (campaign-history.md, ab-tests.md) — an insight without evidence is a hunch, and should be labeled as such until confirmed
- Review this file monthly alongside monthly-retrospectives.md — insights that keep surfacing independently across different retrospectives are the strongest promotion candidates

## Common Mistakes
- Recording an insight as "confirmed" after a single observation — use the Confidence field honestly
- Letting insights accumulate without ever acting on the "Action" field — an insight that never changes anything isn't providing value

## Future Expansion Notes
- A "confirmed" insight that holds across multiple quarters should be promoted into the relevant brand-intelligence document (market-psychology.md, content-system.md, or a persona file) at the next review cycle
