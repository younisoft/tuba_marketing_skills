## Best Practices
- Only log a lesson here once it's been confirmed by more than one instance (a retrospective, a failed campaign, and/or a repeated insight) — this file's value depends on it holding only durable, trustworthy conclusions
- Always state the lesson as a principle, not a narrative — "platform-native pacing affects performance" is usable; "the March campaign didn't do well" is not

## Common Mistakes
- Logging every retrospective takeaway here regardless of durability — this duplicates monthly-retrospectives.md and dilutes the file's purpose
- Failing to close the loop — a lesson that clearly warrants a document update should be tracked to actual promotion, not left "pending" indefinitely

## Future Expansion Notes
- This file, together with confirmed entries in [marketing-insights.md](marketing-insights.md), is the primary input for each Brand Intelligence document's periodic review — at every review cycle, check this file first for anything awaiting promotion
