# Campaign Playbook — Tuba

> **Part of:** [Tuba Advertising Identity System](../ADVERTISING_IDENTITY_GUIDE.md)
> **Builds on:** [advertising-dna.md](advertising-dna.md) · [content-system.md](content-system.md) · [competitor-analysis.md](../knowledge-base/competitor-analysis.md) §10 (seasonal conventions), §11 (anti-patterns)

Every campaign system below follows the same construction: **Objective → Key Message (Guidance Framework) → Primary Channels → Visual Notes → Timing → What Not To Do.** This keeps campaign planning fast and consistent regardless of who's briefing it — human strategist or AI assistant.

---

## 1. Seasonal & Cultural Campaigns

### 1.1 Saudi National Day (يوم الوطني, September 23)

| | |
|---|---|
| **Objective** | Brand/awareness — associate Tuba with belonging and national pride, not a generic discount event |
| **Key message** | Anchor to advertising-dna.md's "a place in the world" — e.g., *"العز في أرضنا، والبيت في قلبها"* (roughly: pride lives in our land, home lives in its heart) |
| **Primary channels** | Snapchat (AR lens, per social-media-playbook.md §4 — the category's proven highest-performing seasonal format), Instagram, YouTube (short brand film) |
| **Visual notes** | Saudi-house icon + desert/palm iconography (iconography.md §4 pairing), Purple-dominant with restrained Coral accent — avoid generic green/white national-colors clichés already owned by every other brand's National Day creative |
| **Timing** | Creative must adapt to **that year's official government-issued National Day theme/slogan** (a confirmed category convention — competitor-analysis.md §10) — check the year's official identity before finalizing concept, do not freelance an unrelated theme |
| **What not to do** | Don't run a generic "we're proud too" post with no distinct idea — every competitor does this; don't discount listings opportunistically in a way that cheapens a national-pride moment |

### 1.2 Founding Day (يوم التأسيس, February 22)

| | |
|---|---|
| **Objective** | Secondary national-pride moment, lower production investment than National Day |
| **Key message** | Heritage-forward — pairs naturally with the Saudi-house icon and "roots" language | 
| **Primary channels** | Instagram, X |
| **Visual notes** | Reuse National Day visual language at reduced scale, distinct headline |
| **Timing** | Feb 22 — plan lightly, don't compete production resource against National Day/Ramadan |
| **What not to do** | Don't treat identically to National Day — distinguish messaging (founding/heritage vs. contemporary national pride) |

### 1.3 Ramadan

| | |
|---|---|
| **Objective** | Sustain engagement through the category's traditionally slower transaction period, without discounting perceived value |
| **Key message** | Financing-term incentives, not price cuts (confirmed category convention — competitor-analysis.md §10): *"لا دفعة أولى حتى الاستلام"* (no down payment until handover) |
| **Primary channels** | All platforms, increased content cadence per the "intensified digital campaign cadence" pattern (competitor-analysis.md §10) |
| **Visual notes** | Warm, gathering-oriented Place & Lifestyle imagery (family iftar/home-gathering framing) — never generic crescent-moon-and-lantern stock cliché; use the existing moon icon (iconography.md) in the brand's own rounded-outline style instead |
| **Timing** | Full month — plan a content arc (early Ramadan: reflection/planning content; mid: financing offers; last 10 days/Eid transition: urgency-free "settle in before Eid" framing) |
| **What not to do** | Don't discount sticker prices directly — protect price/value perception per the documented category convention; don't use artificial countdown pressure (advertising-dna.md §5) |

### 1.4 Eid (Fitr & Adha)

| | |
|---|---|
| **Objective** | Warm, low-pressure brand presence during a family/gathering-focused holiday |
| **Key message** | Belonging-forward, not transactional — *"البيت يتّسع للفرح"* (home makes room for joy) |
| **Primary channels** | Instagram, Snapchat Stories |
| **Visual notes** | Family/gathering Place & Lifestyle content, minimal explicit offer messaging |
| **Timing** | 3-5 days around each Eid — light-touch, goodwill-oriented, not a sales push |
| **What not to do** | Don't run hard-offer campaigns during Eid itself — reserve financing offers for the Ramadan run-up instead |

### 1.5 White Friday (الجمعة البيضاء) — regional Black Friday equivalent

| | |
|---|---|
| **Objective** | Performance-oriented — this is the one moment where direct promotional framing fits cultural expectations |
| **Key message** | Use the Gulf-regional term **"White Friday,"** not "Black Friday" — the region-wide rebrand avoiding the negative connotation of "black" in local context; e.g., *"جمعة بيضاء بعرض واضح"* (a White Friday, a clear offer) |
| **Primary channels** | Performance channels — paid social, search |
| **Visual notes** | Follow visual-system.md §9.6 performance-marketing template — offer-forward, not scene-setting |
| **Timing** | Late November, aligned with regional retail convention |
| **What not to do** | Don't import "Black Friday" branding literally — it reads as a foreign-imported convention, exactly the "imported not indigenous" mistake flagged in competitor-analysis.md §11 |

### 1.6 Back-to-School season (family relocation moment)

| | |
|---|---|
| **Objective** | Rental-focused lead generation — many families finalize a move before the school year starts |
| **Key message** | Practical, time-sensitive-but-not-pressuring: *"استقر قبل أول يوم دراسي"* (settle in before the first day of school) |
| **Primary channels** | Instagram, Facebook (family decision-maker skew per social-media-playbook.md §7), performance search |
| **Visual notes** | Family/neighborhood Place & Lifestyle imagery, school-proximity as a searchable content angle |
| **Timing** | July-August (ahead of the academic year) |
| **What not to do** | Don't fabricate urgency beyond the real, external deadline (school start date) already doing that work |

### 1.7 Summer

| | |
|---|---|
| **Objective** | Off-plan/project campaigns, coastal/vacation-adjacent property content |
| **Key message** | Leverage the sun/sea/wave icon language (iconography.md §3.6) directly — this is the pillar's natural seasonal home |
| **Primary channels** | Instagram, Snapchat |
| **Visual notes** | Coastal Place & Lifestyle photography, sun/wave iconography as decorative pattern (visual-system.md §7.2) |
| **Timing** | June-August |

### 1.8 Winter

| | |
|---|---|
| **Objective** | Cozy/settling-in framing for rental and purchase decisions ahead of year-end |
| **Key message** | Warmth-forward (literal and emotional) — pairs with the brand's core warmth value |
| **Primary channels** | Instagram, YouTube (longer-form "settling in" guidance content) |
| **Timing** | December-January |

## 2. Product & Business Campaigns

### 2.1 Platform launch / major feature launch

| | |
|---|---|
| **Objective** | Awareness + product education |
| **Key message** | Lead with the specific problem solved, not the feature name — e.g., don't announce "we launched Request Property," announce *"You don't have to browse for hours anymore — tell us what you want, and let it come to you."* |
| **Primary channels** | All Tier 1 platforms (social-media-playbook.md §1), PR/earned media if the feature is genuinely novel (Aqar's National Day flyover shows Saudi trade press will cover a bold, well-earned brand moment — competitor-analysis.md §6) |
| **Visual notes** | Full brand-system application — this is a brand-defining moment, invest at visual-system.md's highest production tier |
| **Timing** | Coordinate with product readiness, not calendar convenience |

### 2.2 New property / exclusive listing

| | |
|---|---|
| **Objective** | Lead generation for a specific, often time-sensitive listing |
| **Key message** | Property-description formula (copywriting.md §5.8) + FAL badge (design-components.md) |
| **Primary channels** | Instagram, TikTok, performance search/display |
| **Visual notes** | Photography-led (visual-system.md §6.2), carousel format for multi-room reveal |
| **Timing** | As available; exclusive listings get a short paid-boost window |

### 2.3 Developer partnership / off-plan project launch

| | |
|---|---|
| **Objective** | Qualified buyer demand for a partner developer, brand-safe presentation |
| **Key message** | Professional copy register (copywriting.md §5.2) blended with Place & Lifestyle imagery for the finished-community vision |
| **Primary channels** | LinkedIn (developer/B2B visibility), Instagram (consumer demand generation), dedicated landing page |
| **Visual notes** | visual-system.md §9.8-style structured presentation for developer-facing decks; consumer-facing assets follow standard property showcase templates |
| **Timing** | Phased: announcement → deposit-open window → milestone updates (echoes NHC's ground-level, human-relatable imagery shift — competitor-analysis.md §2 reference case — rather than clinical aerial masterplan renders alone) |

### 2.4 Broker acquisition (B2B)

| | |
|---|---|
| **Objective** | Grow the verified agent/broker network |
| **Key message** | *"شريكك، لا مجرد منصة"* (your partner, not just a platform) — echoes the B2B "partnership" pattern identified in content references (see copywriting.md methodology credit) adapted to broker recruitment specifically |
| **Primary channels** | LinkedIn primary, targeted performance campaigns to broker-adjacent audiences |
| **Visual notes** | Data-forward (lead volume, network credibility) balanced with warm partnership language — copywriting.md §5.2 |
| **Timing** | Always-on, with pushes timed around industry events (e.g., Restatex-style exhibitions — competitor-analysis.md notes Deal app's award recognition came via exactly this kind of event) |

### 2.5 Investment opportunities

| | |
|---|---|
| **Objective** | Position Tuba as a credible investment-decision partner, not just a listings source |
| **Key message** | copywriting.md §5.7 (data first, feeling second) |
| **Primary channels** | LinkedIn, YouTube (deeper data explainers), "دليل طوبة" recurring report (content-system.md §5) as the anchor content property |
| **Visual notes** | Data-visualization-led (visual-system.md §9.8 logic), Coral used only for standout statistics per color hierarchy (visual-system.md §1.7) |

### 2.6 Luxury projects

| | |
|---|---|
| **Objective** | Capture the high-net-worth segment without diluting Tuba's accessible-premium default positioning (advertising-dna.md §10) |
| **Key message** | Restraint-and-specificity luxury copy register (copywriting.md §5.3) — show, don't label |
| **Primary channels** | Instagram (high-production imagery), targeted/invitation-style distribution rather than broad-reach performance ads |
| **Visual notes** | The only campaign context where a more premium, editorial photographic treatment (visual-system.md §6.1's "reserve this register exclusively for luxury campaigns" note) is appropriate — still purple/coral brand-consistent, never a separate sub-brand identity |
| **Note** | Global luxury brands are entering this segment via local licensing partnerships (Sotheby's, Christie's x Wasalt — competitor-analysis.md §1, §6) — Tuba's luxury campaigns should lean on its own brand credibility and specific property quality rather than attempting to compete on borrowed international prestige branding |

## 3. Marketing-Function Campaigns

### 3.1 Lead generation

| | |
|---|---|
| **Objective** | Maximize qualified property requests / listing signups |
| **Key message** | Solve-stage forward (copywriting.md §1) — minimal Hook, immediate value clarity |
| **Primary channels** | Performance search/social, visual-system.md §9.6 template |
| **KPI focus** | Cost per qualified request, request-to-match rate |

### 3.2 Referral campaigns

| | |
|---|---|
| **Objective** | Convert satisfied customers into new-customer sources |
| **Key message** | *"شاركهم الطمأنينة"* (share the peace of mind) — reframes referral as an act of care, not a discount hustle |
| **Primary channels** | Email, WhatsApp (copywriting.md §5.13 register), post-transaction retention touchpoints |
| **KPI focus** | Referral conversion rate, cost per referred lead vs. paid acquisition |

### 3.3 Awareness campaigns

| | |
|---|---|
| **Objective** | Build unaided brand recognition — the purple/coral color-equity opportunity identified in competitor-analysis.md §2 |
| **Key message** | Brand promise (advertising-dna.md §2) stated directly, minimal product-feature detail |
| **Primary channels** | OOH (visual-system.md §9.7), Snapchat/Instagram video, YouTube brand film |
| **KPI focus** | Aided/unaided recall, share of voice |

### 3.4 Brand campaigns

| | |
|---|---|
| **Objective** | Reinforce the full advertising DNA in a single, memorable creative moment (annual or semi-annual) |
| **Key message** | Full six-stage Guidance Framework (copywriting.md §1), highest production value in the calendar |
| **Primary channels** | Multi-platform hero launch, PR support |
| **KPI focus** | Brand lift studies, earned media pickup |

### 3.5 Performance campaigns

| | |
|---|---|
| **Objective** | Direct-response efficiency — leads, app installs, listing submissions |
| **Key message** | Register-shifted per copywriting.md §6 — offer-forward, still warm in word choice |
| **Primary channels** | Paid search, paid social feed/story ad units |
| **KPI focus** | CPL, CPA, ROAS |

## 4. Campaign Governance

- Every campaign brief should state, before production begins: (1) which funnel role it serves (content-system.md §3), (2) which Guidance Framework stages it uses, (3) which pillar (content-system.md §1) it belongs to
- No campaign ships without its FAL/trust element placement decided (design-components.md)
- Seasonal campaigns are locked to the cultural calendar dates in §1 — do not shift Ramadan/National Day creative timing for internal convenience
- Every campaign is conceived bilingually from brief stage, never translated after the fact (advertising-dna.md §9)
