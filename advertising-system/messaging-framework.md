# Messaging Framework — Tuba

> **Part of:** [Tuba Advertising Identity System](../ADVERTISING_IDENTITY_GUIDE.md) — Execution Layer
> **Builds on:** [project-analysis.md §8](../knowledge-base/project-analysis.md) (personas) · [advertising-dna.md](advertising-dna.md)
> **Use:** the Audience × Goal × Emotion × CTA matrix every brief should pull from ([creative-brief-template.md](creative-brief-template.md)). Never write to "everyone" — always route through the specific audience row below.

For each audience: **Pain points → Desired outcome → Core message → Proof → CTA → Emotional angle.** All CTAs are drawn from [copywriting.md §4](copywriting.md); all proof points must be operationally real, never invented (advertising-dna.md §8).

---

## 1. Buyer (general)

| | |
|---|---|
| **Pain points** | Fear of overpaying, fear of legal/paperwork mistakes, information overload |
| **Desired outcome** | A home that's right, bought without regret |
| **Core message** | "You don't have to become an expert to make the right decision — we'll guide you" |
| **Proof** | FAL-licensed listings, request-first matching (no endless browsing), transparent process steps |
| **CTA** | "Request your property" |
| **Emotional angle** | Relief |

## 2. First-Time Buyer

| | |
|---|---|
| **Pain points** | Terrified of getting it wrong (the single largest purchase of their life), doesn't know what questions to ask, distrust of sales pressure |
| **Desired outcome** | Confidence that they didn't miss something important |
| **Core message** | "Every question you're afraid to ask is one we're used to answering" |
| **Proof** | Step-by-step guidance content (content-system.md pillar 1), plain-language FAL/process explainers, human support access |
| **CTA** | "Understand the process" → then "Request your property" |
| **Emotional angle** | Relief + Confidence — this persona needs the *most* Reassure-stage copy of any audience (copywriting.md §1) |

## 3. Luxury Buyer

| | |
|---|---|
| **Pain points** | Generic marketing that doesn't match their taste level; fear of being sold to rather than served; privacy concerns |
| **Desired outcome** | A property that reflects their specific standards, handled with discretion |
| **Core message** | Shown, not told — one specific, unrepeatable property detail beats any luxury adjective (copywriting.md §5.3) |
| **Proof** | Curated/restricted presentation (not broad-reach ad targeting), editorial-quality photography (visual-decision-tree.md §1) |
| **CTA** | "Talk to a guide" (softer, relationship-first register — never a broad "Request now" for this segment) |
| **Emotional angle** | Quiet pride, restraint-signaled confidence — never status-shouting (advertising-dna.md §10) |

## 4. Commercial Buyer

| | |
|---|---|
| **Pain points** | ROI uncertainty, zoning/regulatory complexity, longer decision cycles requiring more stakeholders |
| **Desired outcome** | A defensible, data-backed decision they can justify internally |
| **Core message** | "The numbers, clearly, so you can make the case" |
| **Proof** | Market data, comparable transaction data, regulatory clarity |
| **CTA** | "Understand the process" → direct agent contact |
| **Emotional angle** | Confidence (data-led, per copywriting.md §5.7's investment-copy register) |

## 5. Tenant / Renter

| | |
|---|---|
| **Pain points** | Time pressure, fear of scams/unverified landlords, competing with other applicants |
| **Desired outcome** | A verified, available property, fast |
| **Core message** | "Verified listings, matched to exactly what you asked for" |
| **Proof** | FAL-licensed listings, request-first matching speed |
| **CTA** | "Request your property" |
| **Emotional angle** | Relief (speed-driven — this persona has the *least* patience for a long Guidance Framework arc; compress toward Solve+Decide, copywriting.md §1) |

## 6. Landlord

| | |
|---|---|
| **Pain points** | Fear of bad tenants/vacancy, complexity of rental contracts, wanting fair market pricing without guesswork |
| **Desired outcome** | Reliable income with minimal hassle |
| **Core message** | "List once, reach verified renters, without chasing" |
| **Proof** | Verified renter base, platform-managed process, market-rate pricing data |
| **CTA** | "List with confidence" |
| **Emotional angle** | Relief + Confidence |

## 7. Investor

| | |
|---|---|
| **Pain points** | Distrust of adjective-heavy sales claims, fear of a bad bet, needs to justify the decision with data |
| **Desired outcome** | A quantifiably smart investment, not a persuaded impulse |
| **Core message** | Data first, feeling second (copywriting.md §5.7) — lead with a real, sourced number |
| **Proof** | District-level appreciation data, "دليل طوبة" market report (content-system.md §5), transparent yield calculations |
| **CTA** | Link to full market report → "Understand the process" |
| **Emotional angle** | Confidence — "the kind of number that lets you sleep at night" |

## 8. Family (household decision-makers)

| | |
|---|---|
| **Pain points** | Balancing multiple needs at once (schools, space, safety, budget), decision involves more than one person |
| **Desired outcome** | A home the whole family is settled and happy in |
| **Core message** | "The street your kids grow up on, the kitchen where the family gathers — that's what we're actually looking for" |
| **Proof** | Neighborhood/school-proximity content, family-oriented Place & Lifestyle imagery |
| **CTA** | "Request your property" |
| **Emotional angle** | Belonging |

## 9. Broker / Agent

| | |
|---|---|
| **Pain points** | Lead quality anxiety, needing to look credible to clients, unpredictable income from unreliable lead sources |
| **Desired outcome** | Steady, qualified leads and a professional shopfront that makes them look good |
| **Core message** | "شريكك، لا مجرد منصة" (your partner, not just a platform) |
| **Proof** | Verification/credibility system, lead volume and quality data |
| **CTA** | "Grow with Tuba" |
| **Emotional angle** | Confidence + quiet pride (professional credibility) |

## 10. Developer

| | |
|---|---|
| **Pain points** | Needing qualified buyer demand without damaging project brand perception, long sales cycles for off-plan |
| **Desired outcome** | Brand-safe presentation next to serious buyer intent |
| **Core message** | "Your project, presented the way it deserves, in front of buyers who are actually ready" |
| **Proof** | Structured project-page presentation, buyer-intent data, partnership case studies |
| **CTA** | Partnership inquiry → dedicated developer contact flow |
| **Emotional angle** | Confidence (professional register, copywriting.md §5.2) |

## Cross-Audience Rules

1. **Never write to "everyone."** Every brief (creative-brief-template.md) must name one row from this table as its primary audience — a secondary audience is acceptable, but the Core Message is written for the primary one first.
2. **Compress or expand the Guidance Framework by audience urgency**, not uniformly — Tenants (time-pressured) get a compressed arc; First-Time Buyers (anxious, needs the most reassurance) get the fullest arc.
3. **Proof must match the specific pain point**, not be generic — a Landlord doesn't need to hear about FAL-licensed *listings* (that's a Buyer/Tenant proof point); they need to hear about verified *renters* and contract simplicity.
4. **Luxury and Commercial buyers get lower emotional intensity, higher data/restraint** — this is a deliberate register shift, not a lesser application of brand voice (copywriting.md §5.2-5.3).
