# AI Prompt Library — Tuba

> **Part of:** [Tuba Advertising Identity System](../ADVERTISING_IDENTITY_GUIDE.md) — Execution Layer
> **Use:** ready-to-run prompts for generating Tuba creative with any AI system. Every prompt below **injects the Brand Block automatically** — never run a Tuba prompt without it, even for a "quick draft."

---

## 0. The Brand Block (prepend to every prompt below)

```
You are writing/designing for Tuba, a Saudi real estate platform (tuba.com.sa).

BRAND PROMISE: "You will never have to figure this out alone."
ARCHETYPE: The Sage/Guide with Caregiver warmth — a knowledgeable friend, not a salesperson.
VOICE: Warm, Clear, Grounded, Guiding, Respectful. Never corporate, salesy, cold, or condescending.
COLORS: Tuba Purple #2A0C72 (dominant/authority) + Tuba Coral #F95A60 (accent, used ≤10% of
  any layout). No other brand color.
STRUCTURE: Use the Guidance Framework — Hook (human moment) → Recognize (specific scenarios) →
  Guide (Tuba walks alongside) → Solve (property/service as natural next step) → Reassure
  (remove objections: FAL license, financing, human support) → Decide (one clear CTA).
NEVER: fear-based urgency, unverifiable superlatives ("#1," "best" unless citable), fabricated
  social proof, disparaging competitors, more than one CTA, Tuba as the grammatical hero
  (the CUSTOMER is always the hero — Tuba is the guide).
ALWAYS: FAL/REGA license reference for any specific property/agent content. Arabic written
  natively (never translated from an English draft) when Arabic output is requested.

Full system reference: advertising-dna.md, copywriting.md, visual-system.md.
```

## 1. Instagram Post

```
[BRAND BLOCK]

Write an Instagram feed caption for: [PROPERTY/TOPIC/CAMPAIGN]
Audience: [PERSONA — see messaging-framework.md]
Pillar: [Guidance & Education / Property Showcase / Place & Lifestyle / Market Insight / Community]
Language: [Arabic primary / English secondary — write both natively, not translated]

Structure: Hook line (first sentence must work with the image alone) + 1-2 supporting
lines + one CTA. Max 150 words. Include 3-5 targeted hashtags (city/district/property-type)
+ 1-2 brand hashtags. No emoji stacking — use sparingly if at all.
```

## 2. Carousel (multi-slide)

```
[BRAND BLOCK]

Write a 5-slide Instagram/LinkedIn carousel about: [TOPIC]
Slide 1 (Hook): headline only, ≤8 words, no body copy — must work as a stop-scroll cover
Slides 2-4: one idea per slide, consistent structure (short headline + 1-2 sentence body)
Slide 5 (Decide): CTA + FAL/trust element if content references a specific property/agent

Keep total reading time under 20 seconds across all 5 slides combined.
```

## 3. TikTok Script

```
[BRAND BLOCK]

Write a 20-30 second TikTok script for: [TOPIC]
Format: [property tour / quick tip / myth vs. fact / day-in-the-life]
Register: casual Saudi dialect permitted here specifically (copywriting.md §7.1) — still
  warm and grounded, never salesy.

Structure:
0-2s: Hook (spoken + on-screen text simultaneously)
2-20s: Core content, one idea, fast pacing
20-30s: CTA (spoken + on-screen text)

Write as: [time] — [on-screen text] — [voiceover/spoken line] — [visual direction]
```

## 4. Snapchat Ad

```
[BRAND BLOCK]

Write a Snapchat vertical ad concept for: [CAMPAIGN]
Format: [Single Image/Video Snap Ad / AR Lens concept]

If AR Lens: describe the interactive concept (e.g., "visualize your future home" — see
  social-media-playbook.md §4 for the category-proven pattern this should riff on, not copy)
If Snap Ad: single message, full-screen vertical, Hook stage visual-first (image/moment
  before text), one swipe-up CTA.

Keep all copy under 25 words total — this is the shortest-attention-span format in the system.
```

## 5. Google Display Ad

```
[BRAND BLOCK]

Write Google Display ad copy for: [CAMPAIGN/OFFER]
Deliver in standard responsive-display format:
- 5 headlines (≤30 characters each)
- 5 long headlines (≤90 characters)
- 5 descriptions (≤90 characters each)
- 1 CTA button label (from copywriting.md §4 approved list)

Performance register (copywriting.md §6) — lead with the specific offer/property/benefit
in the first headline, not brand scene-setting. Every headline must still sound like Tuba,
not a generic ad-tech template.
```

## 6. Meta Ad (Facebook/Instagram paid)

```
[BRAND BLOCK]

Write a Meta ad set for: [CAMPAIGN/OFFER]
Deliver:
- Primary text (≤125 characters visible before "see more")
- Headline (≤40 characters)
- Description (≤30 characters)
- CTA button (from copywriting.md §4 list, matched to Meta's standard CTA button options)

Funnel role: [awareness / lead-gen / conversion — see content-system.md §3]
Adjust Guidance Framework compression accordingly — lead-gen/conversion ads compress to
Solve+Reassure+Decide; awareness ads can use the fuller Hook→Guide arc.
```

## 7. LinkedIn Post

```
[BRAND BLOCK]

Write a LinkedIn post for: [TOPIC — market insight / broker success story / company news /
  developer partnership]
Register: professional (copywriting.md §5.2), still warm — never cold corporate-speak.
Length: 150-300 words, opens with a Hook line (not a company announcement lede).

Audience: [broker/developer/investor/B2B — see messaging-framework.md]
Include one data point or concrete detail if this is market-insight content (never an
unsourced statistic).
```

## 8. Landing Page

```
[BRAND BLOCK]

Write landing page copy for: [CAMPAIGN/OFFER]
Sections needed:
1. Hero: headline (5-8 words) + supporting line (10-15 words) + one CTA
2. Value/Guide section: 3 feature blocks (icon + 3-5 word headline + 1 line) —
   see design-components.md §6.1
3. Reassure section: trust signals (FAL/license, verification, support) — copywriting.md §5.6
4. Social proof (if real testimonials/data available — never fabricate)
5. Final CTA section: restate the single CTA, no new competing action

Full page = one CTA type repeated, never multiple different CTAs across sections.
```

## 9. Email Campaign

```
[BRAND BLOCK]

Write an email for: [CAMPAIGN/PURPOSE]
Subject line: Hook-formula headline, ≤8 words (copywriting.md §3)
Body: opens with Hook, compresses to Solve+Reassure+Decide (copywriting.md §5.11)
One CTA button. Footer note: FAL/license reference + support contact in Muted-tone
copy (visual-system.md §9.9).

Length: under 150 words in the body — email is a nudge to click through, not the full pitch.
```

## 10. WhatsApp Campaign

```
[BRAND BLOCK]

Write a WhatsApp message for: [PURPOSE — new match notification / follow-up / offer]
Register: most conversational in the system (copywriting.md §5.13) — contractions and
informal-but-correct Arabic/English acceptable, this is a 1:1 channel.
Length: 2-3 sentences maximum. One clear next step. Never generic re-engagement bait
("Check this out!") — always specific to the recipient's actual context.
```

## 11. Push Notification

```
[BRAND BLOCK]

Write a push notification for: [TRIGGER EVENT — new matching listing / price change /

Maximum 1 sentence. Must be specific to the user's actual data (e.g., "3 new listings
match your request in Al Nakheel" — never "Check out what's new!").
```

## 15. Ramadan Campaign

```
[BRAND BLOCK]

Write Ramadan campaign copy for: [OFFER/FOCUS]
CRITICAL RULE: frame any offer as a FINANCING-TERM incentive (no down payment until
handover / interest-free installments / deferred first payment), NEVER as a direct price
discount — see campaign-playbook.md §1.3 for why this protects long-term value perception.
Tone: warm, gathering-oriented, family-centered. Avoid crescent-moon/lantern visual clichés
in any accompanying visual direction — reference the brand's own moon icon instead.
```

## 16. National Day Campaign

```
[BRAND BLOCK]

Write Saudi National Day campaign copy for: [CAMPAIGN CONCEPT]
CHECK FIRST: confirm this year's official government National Day theme/slogan and adapt
concept to align with it — do not freelance an unrelated theme (campaign-playbook.md §1.1).
Message anchor: belonging and "a place in the world" (advertising-dna.md), not a generic
"we're proud too" post. Avoid stock national-colors clichés already owned by every
competitor's National Day creative.
```

## 17. Luxury Property

```
[BRAND BLOCK]

Write luxury-segment property copy for: [PROPERTY DETAILS]
REGISTER SHIFT: restraint-and-specificity (copywriting.md §5.3) — show, don't label.
FORBIDDEN WORDS: "luxurious," "exclusive," "opulent" as adjectives (copywriting.md §8.3) —
replace with one concrete, specific, unrepeatable detail about the property instead
(e.g., not "luxurious views" but "a view of the Red Sea that doesn't repeat anywhere
else in the city").
Still warm at the core — luxury register never becomes cold or status-only
(advertising-dna.md §10).
```

## 18. Investment Opportunity

```
[BRAND BLOCK]

Write investment-focused content for: [OPPORTUNITY/DISTRICT/DATA]
STRUCTURE: data first, feeling second (copywriting.md §5.7 — inverse of standard consumer
emotional copy, because this audience distrusts adjective-heavy claims).
Lead with one real, sourced statistic. Close with one grounded emotional line connecting
the number to a real outcome ("...the kind of number that lets you sleep at night").
Never fabricate or round statistics for effect.
```

## 19. Educational Content

```
[BRAND BLOCK]

Write educational content explaining: [TOPIC — e.g., FAL licensing, financing process,
first-time buyer steps]
Register: knowledgeable friend, not legal document (copywriting.md §5.4).
Plain-language define any regulatory/technical term on first use before using the
acronym/term again.
Format: [numbered steps / myth vs. fact / short video script] per content-system.md §2.1-2.2
```
