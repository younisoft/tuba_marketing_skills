# Creative Brief Template — Tuba

> **Part of:** [Tuba Advertising Identity System](../ADVERTISING_IDENTITY_GUIDE.md) — Execution Layer
> **Use:** copy this template for every campaign, no matter how small. A one-post social update still fills the short-form version (§2); a seasonal brand campaign fills the full version (§1). An unfilled brief is not a brief — see [campaign-workflows.md §2](campaign-workflows.md).

---

## 1. Full Creative Brief (campaigns, launches, seasonal moments)

```markdown
### Campaign Name
[Short, internal-facing name — e.g., "Ramadan 2027 — Financing Not Fear"]

### Campaign Objective
[One sentence. What business outcome does this serve? Tie to campaign-playbook.md's
campaign type — awareness / lead-gen / conversion / retention / referral / performance]

### Business Goal
[The underlying business driver — e.g., "grow rental listings in Riyadh by X%,"
"support the Q3 developer partnership launch," "sustain engagement during the
seasonal transaction slowdown." Reference project-analysis.md §9-10 business/marketing
goals if this ladders up to one directly.]

### Audience
[Which persona(s) from messaging-framework.md / project-analysis.md §8 does this
target? Name it specifically — "first-time buyers in Riyadh," not "everyone."]

### Core Message
[One sentence — the single idea the audience should walk away with. Must be traceable
to the Brand Promise: "You will never have to figure this out alone" (advertising-dna.md §2)]

### Emotion
[Which brand emotion is this campaign primarily activating? Relief / Belonging /
Confidence / Warmth / Quiet pride — see advertising-dna.md §4]

### Guidance Framework Stage(s) Used
[Full six-stage (Hook→Recognize→Guide→Solve→Reassure→Decide) or compressed
(Hook+Decide only)? See copywriting.md §1]

### CTA
[The single call to action — pulled from copywriting.md §4 or a clear variant]

### Offer (if applicable)
[Specific offer terms — financing structure, not price discount, if this is a
Ramadan-adjacent campaign (campaign-playbook.md §1.3). State exact terms, no vague
"special offer" language.]

### Deliverables
[List every asset needed, by channel and format — e.g.:
- Instagram: 3 feed posts (1:1), 1 carousel (5 slides), daily Stories x 5 days
- TikTok: 2 native videos (9:16, 15-30s)
- Snapchat: 1 AR Lens (if seasonal/brand-tier — social-media-playbook.md §4)
- LinkedIn: 1 long-form post
- Landing page: 1 (if performance-driven)
- Email: 1 send]

### Visual References
[Link or attach any specific inspiration/mood references. Cross-check every reference
against visual-system.md before treating it as directional — an off-brand reference
is not a valid instruction.]

### Success Metrics / KPIs
[Pulled from advertising-playbook.md §9's KPI set, specific to this campaign's
funnel role — e.g., "CPL under X SAR," "3% engagement rate," "500 property requests"]

### Timeline
[Brief approved by: ___
Copy due: ___
Visual due: ___
Review complete: ___
Publish date: ___ (lock to campaign-playbook.md calendar if seasonal)]

### Approvals Required
[Per campaign-workflows.md §7 — which tier: Marketing Manager only / + Brand Owner /
+ Budget Owner]

### Compliance Notes
[FAL/license numbers needed for any specific listing/agent featured — design-components.md §2.1]
```

## 3. Brief Quality Gate

A brief is **not ready** to move to Research/Copywriting until:
- [ ] Objective is one sentence, not a paragraph of ambitions
- [ ] Audience is a named persona, not "everyone" or "all Saudis"
- [ ] Core Message and Emotion trace to advertising-dna.md
- [ ] Exactly one CTA is specified
- [ ] Deliverables list is format- and channel-specific, not "some social content"
- [ ] Success metrics are numeric and time-bound, not "grow awareness"

If any box above is unchecked, return the brief to its author — do not let Research or Copywriting fill these gaps informally, or the campaign loses its accountability trail (see [advertising-playbook.md §8](advertising-playbook.md) on quality gates).
