# Content Review Checklist — Tuba

> **Part of:** [Tuba Advertising Identity System](../ADVERTISING_IDENTITY_GUIDE.md) — Execution Layer
> **Use:** the copy-side counterpart to [design-review-checklist.md](design-review-checklist.md), run at the same stage in [campaign-workflows.md §6](campaign-workflows.md). Every piece of Tuba copy — a headline, a caption, a full campaign script — passes through this before Approval.

---

## 1. Headline

- [ ] 5-8 words (copywriting.md §3)
- [ ] Uses one formula, or at most two layered, from the approved list (copywriting.md §3)
- [ ] Would survive standalone on an OOH board with zero supporting copy (visual-system.md §9.7 test)
- [ ] Not a direct feature statement ("New 3BR villa available") without an emotional frame around it

## 2. Opening Hook

- [ ] Names a human moment (dream or worry) before mentioning Tuba, the property, or a feature (copywriting.md §1, Stage 1)
- [ ] Specific, not abstract — a real image/scenario, not a vague adjective ("beautiful," "amazing")
- [ ] Lands within the first line/first 3 seconds (social-media-playbook.md hook rules)

## 3. Body

- [ ] Follows the appropriate Guidance Framework stages for this content's length/purpose (copywriting.md §1)
- [ ] One idea per sentence — no "and" joining two unrelated claims (copywriting.md §2.2)
- [ ] Sentence length varies deliberately — not three same-length sentences in a row
- [ ] Any regulatory/technical term (FAL, REGA, etc.) is plain-language defined on first use (copywriting.md §5.4)
- [ ] Data/statistics cited with a source, never presented as unsourced fact (copywriting.md §5.7)

## 4. CTA

- [ ] Exactly one CTA (copywriting.md §4, advertising-checklists.md §11)
- [ ] Specific to the action, not generic ("Contact us today" is forbidden — copywriting.md §8.3)
- [ ] Matches the funnel role of this content (content-system.md §3) — e.g., lead-gen content CTAs to a request/contact action, not a generic "learn more"

## 5. Grammar

- [ ] No spelling or grammatical errors in either language
- [ ] Consistent editorial style applied (copywriting.md §7.3) — spec order (bed→bath→area→price), currency format, numeral system
- [ ] Arabic diacritics used only where genuinely needed for clarity (not decoratively, not omitted where ambiguity would result)

## 6. Saudi Arabic Suitability

- [ ] Register matches context — Saudi/Gulf-inflected Modern Standard Arabic for brand copy, casual dialect only where explicitly permitted (short-form TikTok/Snapchat captions — copywriting.md §7.1)
- [ ] Direct address (أنت/ك) used as the dominant pronoun, not distancing plural/formal address ("عملاءنا الكرام" is forbidden — copywriting.md §8.3)
- [ ] Western numerals used for prices/specs (visual-system.md §2.4)
- [ ] No literal word-for-word translation artifacts from an English draft — reads as natively written (advertising-dna.md §9)
- [ ] Cultural references (seasonal, family, place) are accurate to actual Saudi context, not generic pan-Arab or imported phrasing

## 7. Brand Tone

- [ ] Reflects all five voice adjectives: Warm, Clear, Grounded, Guiding, Respectful (advertising-dna.md §11)
- [ ] Customer is the grammatical subject/hero, Tuba is the guide (advertising-dna.md §3, Principle 1)
- [ ] No forbidden expressions present (copywriting.md §8.3 — cross-check the full table)
- [ ] Tone register matches the content's context (copywriting.md §2.1 — B2B vs. consumer vs. support vs. campaign)

## 8. Psychological Triggers

- [ ] Any persuasion technique used is drawn from the approved list (psychological-triggers.md) and used within its stated guardrails
- [ ] No fear-based urgency or fabricated scarcity (advertising-dna.md §5)
- [ ] No comparison-shaming or guilt framing (copywriting.md §9)
- [ ] Social proof (if used) is real, not fabricated (copywriting.md §9)

## 9. Persuasiveness

- [ ] Objection is pre-empted before the reader has to voice it, where relevant (Reassure stage — copywriting.md §1)
- [ ] Specificity used over abstraction (a real number/detail beats a vague adjective — copywriting.md §9)
- [ ] Reads as confident but not salesy — no exclamation-point inflation, no superlative stacking

## 10. Readability

- [ ] Passes a "read aloud" test — does it sound like a person talking, not a document?
- [ ] Appropriate length for format (see social-media-playbook.md per-platform structure rules)
- [ ] No jargon left unexplained for a general consumer audience (B2B content may retain more technical language — copywriting.md §5.2)

## 11. Platform Optimization

- [ ] Caption/copy length matches the platform's norm (social-media-playbook.md, per-platform "post structure" rules)
- [ ] Hashtags (where applicable) match platform-specific strategy and count (social-media-playbook.md)
- [ ] Register matches platform (formal LinkedIn vs. casual TikTok — copywriting.md §2.1)

## 12. SEO (where relevant — blog/landing page/YouTube content only)

- [ ] Primary keyword/topic appears in headline and first paragraph naturally (never keyword-stuffed)
- [ ] Meta description (if applicable) is a Hook-formula sentence, not a generic summary
- [ ] Headers (H2/H3) used to structure long-form content logically, matching visual-system.md §2.3 hierarchy
- [ ] Alt text written for all images — descriptive, not keyword-stuffed, genuinely useful for accessibility

---

## Quick Fail Conditions (automatic return, no partial credit)

Any one of these sends the content back to Copywriting regardless of how well everything else scores:
- Contains a forbidden expression (copywriting.md §8.3)
- Makes an unverifiable superlative claim
- Contains fear-based urgency or fabricated scarcity/social proof
- Is a literal translation rather than natively-written Arabic
- Has more than one CTA
- Fails to plain-language-define a regulatory term on first use in educational content
