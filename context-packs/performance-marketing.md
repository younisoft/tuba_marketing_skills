## The Core Difference from Brand Content

Performance contexts reward **clarity-first**, not scene-setting. The Guidance Framework compresses: lead with Solve-stage specifics (price, location, spec) within the first line, keep Hook to a fragment, always end with one strong CTA. Clarity-first is not voice-abandoning — warmth still shows in word choice even in a four-word headline.

## Format Specs

| Format | Rule |
|---|---|
| Standard IAB display (300×250, 728×90, 160×600, 320×50) | At 320×50: mark + 3-4 word headline + CTA only — never attempt full hierarchy |
| Google Display responsive | 5 headlines (≤30 chars), 5 long headlines (≤90 chars), 5 descriptions (≤90 chars), 1 CTA |
| Meta ad | Primary text ≤125 chars visible, headline ≤40 chars, description ≤30 chars |

## Copy Pattern

```
[Offer/spec] + [Location] + [One differentiator]
```
Example: "3BR villa, Al Nakheel — FAL-verified, ready now"

## Urgency & Scarcity — Hard Guardrails

**Only ever use urgency tied to a real, checkable deadline.** No countdown-timer fabrication, no "only 2 left" unless literally true. Real-deadline examples: a genuine financing-offer expiration, an actual school-year start date, a real limited-unit-count phase. Full guardrail table: [psychological-triggers.md §3-4](../advertising-system/psychological-triggers.md).

## CTA Rule

One CTA, high-contrast (Coral fill), always present — never the first thing cut for space at small sizes.

## Trust Signal — Never Skip

Even in a compressed performance layout, the FAL badge stays present — trust signaling is not sacrificed for space.

## KPI Reference

| Funnel role | Primary KPIs |
|---|---|
| Lead generation | Cost per qualified request, request-to-match rate, form completion |
| Conversion | CPL, CPA, listing-to-contact rate |
| Performance (paid) | CTR, CPM, ROAS, CPA |

Category benchmark (directional): 1-2% baseline engagement is common industry starting point; 4-7% achievable in 90 days with disciplined execution.

## A/B Testing Rule

Test **one variable at a time** — headline OR image OR CTA, never multiple simultaneously. A test changing two things produces a result but no reusable learning.
