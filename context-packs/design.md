# Context Pack — Design

## Color (the fast version)

| Token | Hex | Rule |
|---|---|---|
| Purple | `#2A0C72` | Dominant, ~30% |
| Coral | `#F95A60` | Accent, ≤10%, ≤2 uses per composition |
| Canvas | `#F7F5FC` | ~60% ground |
| Ink (text) | `#1A0E35` | Never pure black |
| Muted (secondary text) | `#6B5F8A` | Never pure grey |

Never Purple-on-Coral or Coral-on-Purple body text. Gradient (Purple→Coral, 135°) reserved for sparse-headline hero moments — never behind dense copy.

## Typography

Almarai (Arabic) + Plus Jakarta Sans (Latin). Max 3 sizes per asset. Arabic runs 1-2px larger than Latin at equal weight.

## Spacing & Radius

8px scale (4·8·16·24·32·48·64·96). Buttons 12px/pill, cards 16-20px, badges full pill. No sharp 0px corners on brand-owned containers.

## Shadows

Purple-tinted only: `0 2px 16px rgba(42,12,114,.09)` (rest) → `0 8px 36px rgba(42,12,114,.15)` (hover).

## Routing Logic (visual-decision-tree.md — pick the branch, follow it)

| Campaign type | Layout | Photography | CTA style |
|---|---|---|---|
| Luxury | More negative space, editorial | Premium, still warm | Secondary/outline button |
| Property Showcase (standard) | Card/carousel | Straightened, warm light | Primary Coral pill |
| Investment/Data | Stat block/row | Minimal, data-led | Link to report |
| Educational | Carousel/infographic | Flat illustration preferred | "Understand the process" |
| B2B | LinkedIn-native/deck | Professional, ground-level + masterplan | "Grow with Tuba" |
| Seasonal | Per-occasion icon pairing | Per-occasion | Standard |
| Performance | Compressed, offer-forward | High-clarity, no scene-setting | Mandatory Coral CTA |
| Awareness/Brand | OOH-legibility standard | Highest production value | Soft brand CTA |

## Icons — Quick Rules

- Purple only, never Coral
- One treatment (filled OR outline) per composition
- Real estate family + desert/lifestyle family — pair them for "home + place" storytelling (iconography.md §4)
- Stroke ≈1.72px, soft miter, proportional scaling only

## Photography Direction

Real Saudi context, warm/golden-hour lighting, straightened verticals, negative space for text, real people in real moments — never generic stock or staged handshakes.

## Component Basics

| Component | Rule |
|---|---|
| Property card | 16-20px radius, price bold, FAL badge bottom corner |
| Primary CTA | Coral fill — never Purple-filled |
| Trust badge | Pill, Purple 8 bg, verified checkmark |
| Status tags | Coral 20 bg for New/Featured/Off-Plan |

## Motion (when needed)

Soft ease-in-out only, never linear/bouncy. Logo may scale/fade, never spin/distort. Icon draw-on for outline icons. Gradient may animate slowly (8-12s loop), never flash.
