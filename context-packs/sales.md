## Broker Sub-Personas (quick map)

| Sub-type | Core need | Best CTA |
|---|---|---|
| Independent/Solo Agent | Consistent, proven lead flow | "Grow with Tuba" |
| Agency-Affiliated Agent | Individual visibility within a larger roster | "Grow with Tuba" |
| Boutique/Luxury Broker | Curated, discreet HNW demand | "Talk to a guide" (relationship-first) |
| New/Junior Agent | Build a client base from zero | "Grow with Tuba" |
| High-Volume Agent | Scale/efficiency tools | "Grow with Tuba" |

**Universal broker objection:** "I've been burned by platforms that promised leads and delivered impressions" — always answer with real lead-quality proof, not brand messaging alone.

## Developer Sub-Personas (quick map)

| Sub-type | Core need |
|---|---|
| Large Master-Plan Developer | Brand-safe presentation at scale, phased buyer-intent data |
| Boutique Developer | Visibility despite smaller scale |
| Off-Plan Specialist | Overcoming off-plan trust hesitation |
| Government-Linked/Institutional | Regulatory/public-trust alignment, most formal register |

## Copy Register for B2B

Professional (copywriting.md §5.2) — less emotional scene-setting than consumer copy, but never cold. Lead with outcome: *"Verified leads. Licensed platform. Your shopfront, done right."*

## Content Rules

- **Broker content:** case-study format, real lead-volume/quality numbers, agent-as-hero imagery (real people, not stock handshakes)
- **Developer content:** data-forward partnership decks; ground-level human imagery paired with any masterplan/aerial shots — never aerial-only (the NHC rebrand lesson, [competitor-analysis.md §2](../knowledge-base/competitor-analysis.md))
- **Off-plan specifically:** foreground escrow/regulatory trust mechanisms — this audience needs more Reassure-stage content than ready-built

## Partnership Journey

Longer B2B cycle, multiple stakeholders — requires case studies and structured proposals, not a single ad. Government-linked developers need market-register formality ([market-language.md](../brand-intelligence/market-language.md)), not consumer warmth.

## Trust Signal for B2B

Lead-quality/verification credibility point is mandatory in any broker-facing copy — this audience's #1 objection is lead quality, not price.
