## The Guidance Framework (use for any substantial copy)

```
1. Hook — name a human moment (dream or worry), before mentioning Tuba
2. Recognize — 2-3 branching scenarios ("or," not "and")
3. Guide — "we'll help you..." register, Tuba walks alongside
4. Solve — the property/service as the natural next step
5. Reassure — remove the specific objection (FAL, financing, human support)
6. Decide — one clear CTA, stated with quiet confidence
```

Short copy (a caption, a notification) compresses to Hook + Decide only.

## Voice — Five Adjectives

**Warm · Clear · Grounded · Guiding · Respectful.** The customer is always the grammatical hero; Tuba is the guide, never the subject when the customer's life could be instead.

## Sentence Rules

- Headlines: 5-8 words. Supporting lines: 10-15 words.
- One idea per sentence — split any sentence joined by "and" across two unrelated claims.

## Headline Formula Quick Table

| Formula | Pattern | Use when |
|---|---|---|
| The Open Dream | "Ever imagined [dream]?" | Emotional Hook stage |
| The Quiet Reveal | "[Simple truth] in [N words]" | Trust/authority |
| The Binary Choice | "[A] or [B]?" | Engage/Recognize stage |
| The Name Play | Wordplay on Tuba/طوبة | Brand campaigns |
| The Concrete Number | Emotional phrase + one real number | Investment, Sakani-pattern campaigns |
| The Direct Address | "You" placed first | Any Hook |

Full library + why each works: [headline-library.md](../brand-intelligence/headline-library.md).

## CTA Rule

**Exactly one CTA per asset, always.** Pull from category (Awareness/Lead-Gen/Consultation/Booking/WhatsApp/Investment/Broker/Referral/Luxury/etc.) in [cta-library.md](../brand-intelligence/cta-library.md) — never a generic "Contact us today."

## Forbidden (hard rules — CLAUDE_SYSTEM_PROMPT.md §7)

- Fear-based urgency, fabricated scarcity/deadlines
- Unverifiable superlatives ("#1," "the best") unless independently citable
- Fabricated social proof (reviews, testimonials, user counts)
- Disparaging competitors by name or implication
- More than one CTA
- Generic AI phrases: "unlock," "elevate," "seamless," "revolutionize," "game-changing," "unparalleled," "in today's world," "look no further"

## Preferred Vocabulary (sample — full lists in copywriting.md §8, emotional-keywords.md)

**EN:** together, guide, verified, relief, ready, real, here for you
**AR:** معاً، نساعدك، بثقة، واضح، موثّق، اطمئن، بيتك

## Language Rules

- Arabic is written **natively**, never translated from a finished English draft — Saudi/Gulf-inflected MSA, direct address (أنت/ك)
- Western numerals for prices/specs, not Eastern Arabic-Indic
- Never justify Arabic body text in narrow columns
- Plain-language define any regulatory term (FAL, REGA) on first use

## Register by Context

| Context | Register |
|---|---|
| Performance/paid ad | Direct, offer-forward, compressed Guidance Framework |
| Brand/awareness | Full six-stage framework |
| B2B (agents/developers) | Professional, less emotional scene-setting, still warm |
| Luxury | Restraint + specificity — never label ("luxurious"), show one real detail instead |
| Investment | Data first, feeling second (inverse of consumer copy) |

## Trust Copy Pattern (use constantly)

Never "we're trusted." Always the specific mechanism: *"Every listing carries a FAL license number you can verify yourself."*
