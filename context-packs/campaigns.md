## Seasonal Calendar Rules (hard rules, not suggestions)

| Occasion | Rule |
|---|---|
| **National Day** (Sept 23) | Check that year's official government theme/slogan FIRST — never freelance a new concept |
| **Founding Day** (Feb 22) | Heritage-forward, lower production scale than National Day |
| **Ramadan** | Financing-term incentives (no down payment until handover, deferred installments) — **never direct price cuts** |
| **Eid** | Family/gathering imagery, minimal explicit offer messaging |
| **White Friday** | Use "White Friday," never "Black Friday" (regional convention) |
| **Back-to-School** | Real-deadline urgency only (actual school start date) |
| Summer / Winter | Coastal/lifestyle vs. cozy/settling-in framing |

## Campaign Types

**Product/Business:** Platform launch · New property/exclusive listing · Developer partnership · Broker acquisition · Investment opportunities · Luxury projects
**Marketing-function:** Lead generation · Referral · Awareness · Brand · Performance

## The Brief (always fill before producing anything)

```
Objective (1 sentence) → Audience (named persona, never "everyone") →
Core Message → Emotion → Guidance Framework stage(s) → CTA →
Offer (if any) → Deliverables → Success Metrics → Timeline → Approvals
```
Full template: [creative-brief-template.md](../advertising-system/creative-brief-template.md).

## Audience Quick Map (full detail: messaging-framework.md, persona files)

| Audience | Lead emotion | Best CTA |
|---|---|---|
| First-Time Buyer | Relief + Confidence | "Understand the process" → "Request your property" |
| Family | Belonging | "Request your property" |
| Investor | Confidence (data-first) | Link to market report |
| Luxury Buyer | Quiet pride (restrained) | "Talk to a guide" |
| Tenant/Renter | Relief (speed) | "Request your property" |
| Broker/Agent | Confidence + partnership | "Grow with Tuba" |
| Developer | Confidence (professional) | Partnership inquiry |

## Process Flow (10 stages — full Mermaid diagrams in campaign-workflows.md)

```
Idea → Research → Brief → Copy → Design → Review → Approval → Publishing → Optimization → Reporting → Knowledge Capture
```

**Quality gate:** design score ≥8/10 (no dimension <6) AND content checklist fully passes before Approval.

## Campaign Governance (hard rules)

- Every campaign traces to the Brand Promise — if it could run unchanged for a competitor, rewrite it
- FAL/trust element placement decided before shipping
- Seasonal timing locked to real calendar dates — never shifted for convenience
- Conceived bilingually from brief stage, never translated after

## Approval Tiers

| Tier | Examples | Sign-off |
|---|---|---|
| Standard | Routine post, single listing | Marketing Manager |
| Seasonal/Brand | National Day, Ramadan, brand campaign | + Brand Owner |
| Paid above threshold | Performance spend | + Budget Owner |
| Regulatory-sensitive | Specific FAL/financing claims | + Compliance |
