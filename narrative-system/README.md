# Narrative System — README

> **This folder is Tuba's creative DNA.** Everything in `advertising-system/`, `brand-intelligence/`, and `context-packs/` tells you *how to execute* a Tuba asset correctly. This folder tells you *how Tuba thinks, feels, sees, and remembers* — the layer that makes execution feel inevitable instead of arbitrary.

---

## What This Is, and Isn't

This is **not** a rulebook. It has no checklists, no hex codes, no CTA formulas, no character counts. Those live in [`advertising-system/`](../advertising-system/) and [`brand-intelligence/`](../brand-intelligence/), and this folder never repeats them — it references them and builds one level above them.

This **is** the narrative universe the approved BIG IDEA — **[Everyone's Tuba Is Different](../BIG_IDEA_PLATFORM.md)** — expands into. Where `BIG_IDEA_PLATFORM.md` discovered *the idea*, this folder builds *the world the idea lives in*: its philosophy, its people, its symbols, its way of seeing, its way of speaking, and the raw story material hundreds of future campaigns can draw from without ever having to reinvent the brand.

Start at the root document, **[`../NARRATIVE_SYSTEM.md`](../NARRATIVE_SYSTEM.md)** — the highest-level creative document in the entire repository. Every file below is one layer of depth beneath it.

## Governing Rules

1. **Never contradict `advertising-dna.md`.** This folder is downstream of it, the same way every document in `advertising-system/` is (`advertising-dna.md`: *"if a piece of creative contradicts this document, the creative is wrong, not this document"* — that inheritance rule applies here too).
2. **Never re-explain the BIG IDEA.** `BIG_IDEA_PLATFORM.md` already made the case for *why* "Everyone's Tuba Is Different" won. This folder's job is to make it feel inevitable through world, not to keep arguing for it.
3. **Never duplicate execution-layer documents.** If a fact already lives in `advertising-system/copywriting.md`, `visual-system.md`, `brand-intelligence/storytelling-framework.md`, or `emotional-keywords.md`, this folder links to it instead of restating it.
4. **Optimize for a decade, not a quarter.** Nothing in this folder should reference a specific platform, format, or year. That belongs in `advertising-system/` and `context-packs/`.
5. **This produces no assets.** No headlines, no posts, no campaigns exist inside this folder. It produces *material* — the next phase produces executions from it.

## Owner & Review

**Owner:** Brand Owner / Executive Creative Director function.
**Review frequency:** annually — this folder should be the most stable, slowest-changing layer in the entire platform, alongside `brand-intelligence/creative-principles.md`. If something here needs frequent revision, it was written at the wrong altitude.
