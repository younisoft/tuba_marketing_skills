## 1. Independent/Solo Agent

| Field | Detail |
|---|---|
| **Goals** | Consistent, qualified lead flow; looking credible to clients without a large agency backing them |
| **Fears** | Unpredictable income; being outcompeted by agency-backed agents with more resources |
| **Pain points** | Limited marketing budget/skill of their own; needs the platform to do the credibility-building work |
| **Questions** | "How many real leads will this actually bring me?" "What does it cost, and is it worth it?" |
| **Objections** | "I've been burned by platforms that promised leads and delivered impressions" |
| **Emotional triggers** | Confidence, professional pride (quiet pride, per emotional-keywords.md §5) |
| **Buying journey** *(acquisition journey)* | Price/ROI-sensitive, needs proof before commitment — case studies and real numbers matter more than brand messaging alone |
| **Content preferences** | Lead-quality proof, success-story content, clear pricing/value explanation |
| **Visual preferences** | Straightforward, credibility-focused (professional headshot integration, design-components.md §1.2) |
| **Best CTA** | "Grow with Tuba" |

## 2. Agency-Affiliated Agent

| Field | Detail |
|---|---|
| **Goals** | Standing out within a larger agency roster; personal brand-building alongside agency affiliation |
| **Fears** | Being just another face in a large listing pool; agency politics affecting lead distribution |
| **Pain points** | Needs a platform presence that reflects individual credibility, not just agency-level branding |
| **Questions** | "Can I build my own visibility here, not just get folded into the agency's presence?" |
| **Objections** | "My agency already has its own tools/relationships" |
| **Emotional triggers** | Confidence, individual recognition |
| **Buying journey** | Often influenced by agency-level decisions, but personally motivated by individual visibility/lead-quality |
| **Content preferences** | Individual agent-spotlight content (content-system.md §4.8) |
| **Visual preferences** | Personal branding elements within the platform's system (circular headshot crop, badge — design-components.md §1.2, §2.2) |
| **Best CTA** | "Grow with Tuba" |

## 3. Boutique/Luxury Broker

| Field | Detail |
|---|---|
| **Goals** | Access to genuinely qualified high-net-worth buyer demand; brand association that matches their own premium positioning |
| **Fears** | Being associated with a mass-market platform in a way that undercuts their own luxury credibility |
| **Pain points** | Most listings platforms aren't built for discretion or curated presentation |
| **Questions** | "How does Tuba handle discretion and curated distribution for high-value listings?" |
| **Objections** | "This platform isn't built for the segment I work in" |
| **Emotional triggers** | Quiet pride, restraint (luxury-framework.md) |
| **Buying journey** | Relationship-first, slower, values direct conversation over self-serve signup |
| **Content preferences** | Luxury-segment case studies, curated/invitation-style program details (campaign-playbook.md §2.6) |
| **Visual preferences** | Editorial-quality, restrained (visual-decision-tree.md §1) |
| **Best CTA** | "Talk to a guide" (relationship-first, not a broad signup CTA) |

## 4. New/Junior Agent

| Field | Detail |
|---|---|
| **Goals** | Build a client base and reputation from a standing start |
| **Fears** | Not being taken seriously due to inexperience; struggling to compete with established agents |
| **Pain points** | Lacks an existing network/lead source; needs the platform to substitute for years of built relationships |
| **Questions** | "Can this actually help me build a client base from zero?" "Is the barrier to entry realistic for someone new?" |
| **Objections** | "Experienced agents will always get the good leads first" |
| **Emotional triggers** | Confidence, encouragement (without being condescending — copywriting.md's "respectful" voice adjective applies here specifically) |
| **Buying journey** | Highest need for onboarding/education support among broker personas |
| **Content preferences** | Onboarding guidance, "getting started" educational content |
| **Visual preferences** | Approachable, encouraging, clear |
| **Best CTA** | "Grow with Tuba" |

## 5. High-Volume Agent

| Field | Detail |
|---|---|
| **Goals** | Efficiency and scale — managing many listings/leads without the process breaking down |
| **Fears** | A platform that can't handle their volume, or that buries their listings among too many competitors |
| **Pain points** | Needs tools/visibility that scale, not just a single-listing experience |
| **Questions** | "Can this handle my actual volume?" "What does performance/analytics visibility look like at scale?" |
| **Objections** | "I've outgrown what most platforms can offer" |
| **Emotional triggers** | Confidence, efficiency-driven (low emotional intensity, per emotional-keywords.md §8) |
| **Buying journey** | Data/ROI-driven, evaluates the platform as a business tool, not a brand relationship |
| **Content preferences** | Analytics/performance-data content, case studies with real volume numbers |
| **Visual preferences** | Data-forward, minimal decoration |
| **Best CTA** | "Grow with Tuba" |

## Best Practices
- Broker/agent content should always answer the specific credibility or lead-quality question in the persona's Questions row — generic "join our platform" messaging under-performs across every broker sub-type
- Luxury/boutique broker outreach should never be lumped into mass broker-acquisition campaigns (campaign-playbook.md §2.4) — use a distinct, relationship-first approach

## Common Mistakes
- Marketing to all agents with volume/scale messaging when a meaningful share are solo/boutique operators who need credibility and quality messaging instead
- Under-serving New/Junior Agents with content that assumes an existing client base
