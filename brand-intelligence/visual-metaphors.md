# Visual Metaphors — Tuba

> **Layer:** Brand Intelligence (static)
> **Purpose:** the symbolic visual language behind Tuba's concepts — which image/shape/motif represents which idea, and when to reach for it. Extends the existing icon vocabulary ([iconography.md](../advertising-system/iconography.md)) into a strategic metaphor system for campaign concepting, not just literal icon usage.
> **Owner:** Design lead
> **Review frequency:** semi-annually
> **Related documents:** [iconography.md](../advertising-system/iconography.md), [visual-system.md §6-7](../advertising-system/visual-system.md), [storytelling-framework.md](storytelling-framework.md)

---

## 1. Growth

**Metaphor:** upward, organic movement — a rising sun, a growing plant, an expanding horizon line.
**When to use:** investment content, market-insight stat reveals, broker/developer growth messaging.
**Existing asset match:** sun iconography (iconography.md §3.6), stat block count-up animation (visual-decision-tree.md §3).
**Avoid:** literal bar-chart-arrow clichés — Tuba's growth metaphor should stay organic/place-rooted, not generic corporate infographic imagery.

## 2. Security

**Metaphor:** enclosure with warmth — the hut/roof shape from the logo mark itself, a protected courtyard (حوش), gathered light.
**When to use:** trust/compliance content, First-Time Buyer and Retiree persona content (buyer-personas.md §1, §6).
**Existing asset match:** the logo mark's hut silhouette (current-brand-analysis.md §2), rounded shape language throughout (visual-system.md §3.3).
**Avoid:** literal padlock/shield clichés unless paired with the actual verified-badge icon (design-components.md §2.1) — a bare padlock icon reads as generic fintech, not Tuba.

## 3. Opportunity

**Metaphor:** an open door, a horizon, a path forward — movement toward, not arrival yet.
**When to use:** investment and first-time-buyer content, Hook-stage imagery (copywriting.md §1).
**Existing asset match:** direction/wayfinding icons (iconography.md §3.2), open negative space in luxury/aspirational photography (visual-system.md §6.1).

## 4. Home

**Metaphor:** the hut/roof shape (logo mark), warm interior light, a gathered table.
**When to use:** universally — this is the brand's most central metaphor, present in nearly every property/family/community asset.
**Existing asset match:** the logo mark itself (current-brand-analysis.md §2), house icon family (iconography.md §3.1).

## 5. Investment

**Metaphor:** compounding/layering — stacked, deliberate structure (echoing the logo's circle-built construction discipline), not a generic dollar-sign or chart icon.
**When to use:** investor-persona content, alongside real data visualization (design-components.md §3).
**Existing asset match:** stat block/stat row components (design-components.md §3.1-3.2).

## 6. Trust

**Metaphor:** the checkmark-in-circle (verified badge), the QR-code-as-proof device (design-components.md §2.1) — trust as something *checkable*, not just asserted.
**When to use:** every commerce-adjacent asset (advertising-dna.md §8).
**Existing asset match:** the FAL/Verified badge system (design-components.md §2.1-2.2).

## 7. Luxury

**Metaphor:** negative space itself — what's *not* shown communicates as much as what is. Restraint as the visual signal, not ornamentation.
**When to use:** luxury-segment campaigns only (campaign-playbook.md §2.6).
**Existing asset match:** visual-decision-tree.md §1's luxury layout guidance (more negative space, editorial spacing).
**Avoid:** gold/ornate decorative motifs — inconsistent with Tuba's restrained luxury register (copywriting.md §5.3, luxury-framework.md).

## 8. Community

**Metaphor:** the gathered table, overlapping/interlocking circles (echoing the logo's circle-construction logic to represent people connecting), a shared majlis.
**When to use:** community/customer-success content (content-system.md pillar 4), Eid/Ramadan seasonal content (campaign-playbook.md §1.3-1.4).
**Existing asset match:** none dedicated yet — see asset-library.md §3 (People) gap.

## 9. Future

**Metaphor:** the horizon, a rising sun, an unfinished/growing line (echoing off-plan/under-construction storytelling).
**When to use:** off-plan/project campaigns, aspirational Hook-stage content.
**Existing asset match:** sun iconography (iconography.md §3.6), the "future aspiration" emotional register (psychological-triggers.md §10).

## 10. Movement

**Metaphor:** waves (smooth and sharp variants), wind-direction icons — literal existing brand assets already carrying this meaning.
**When to use:** relocation-triggered content (campaign-playbook.md §1.6), "journey" framing in customer-journey.md narratives.
**Existing asset match:** waves icon family, direction icons (iconography.md §3.2, §3.6).

## 11. Progress

**Metaphor:** the numbered-step visual language already used in design-components.md's process content, icon draw-on animation (visual-system.md §8) as literal "building toward completion."
**When to use:** educational/process content (content-system.md pillar 1), off-plan construction-progress content (developer-personas.md §3).
**Existing asset match:** feature block/step components (design-components.md §6.1), icon draw-on motion (iconography.md §1.2).

## 12. Saudi Identity

**Metaphor:** the desert/lifestyle icon set as a whole — sun, palm, mountains, Saudi-house silhouette — Tuba's most distinctive and currently under-leveraged metaphor system (competitor-analysis.md §9, project-analysis.md §20).
**When to use:** National Day/Founding Day content (campaign-playbook.md §1.1-1.2), Place & Lifestyle pillar broadly.
**Existing asset match:** the full desert/lifestyle icon family (iconography.md §3.6) — see iconography.md §4's Home+Place pairing logic for the combinatorial version of this metaphor.

## Best Practices
- Choose one primary metaphor per asset — layering multiple metaphors (e.g., growth + security + community icons all in one composition) creates visual noise, mirroring the "one idea per sentence" copy rule (copywriting.md §2.2)
- Always route a metaphor through an *existing* brand asset (icon, shape, color) before inventing new imagery — see asset-library.md for what currently exists

## Common Mistakes
- Reaching for generic real-estate/fintech stock metaphors (padlocks, dollar signs, generic bar charts) instead of Tuba's own established shape language
- Using the Luxury (restraint) metaphor inconsistently — mixing ornate decorative elements into an otherwise restrained luxury layout
