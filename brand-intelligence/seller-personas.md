## 1. Individual Homeowner Seller

| Field | Detail |
|---|---|
| **Goals** | A fair price, a smooth process, minimal disruption to daily life during the sale |
| **Fears** | Undervaluing the property; a buyer falling through late in the process; a bad-faith buyer or agent |
| **Pain points** | Doesn't know true market value; unsure how to present the property competitively |
| **Questions** | "What's my property actually worth right now?" "How do I know the agent is getting me the best deal?" |
| **Objections** | "I'd rather just work with a family friend who's an agent" |
| **Emotional triggers** | Confidence (fair value), Relief (smooth process) |
| **Buying journey** *(selling journey)* | Starts with informal value-checking (word of mouth, casual research), moves to formal listing once confident in pricing |
| **Content preferences** | Pricing/valuation guidance content, "what affects your property's value" educational content |
| **Visual preferences** | Clear, credible data presentation (design-components.md §3) |
| **Best CTA** | "List with confidence" |

## 2. Investor-Seller ("Flipper" / short-hold seller)

| Field | Detail |
|---|---|
| **Goals** | Maximize return on a shorter hold period; fast, efficient transaction turnaround |
| **Fears** | A slow sale eating into returns; underpricing due to time pressure |
| **Pain points** | Needs market timing data, not just a listing service |
| **Questions** | "What's the fastest realistic timeline to sell at a good price?" "How's the market trending right now?" |
| **Objections** | "I need more certainty on timing before I commit to a platform" |
| **Emotional triggers** | Confidence (data-first, per copywriting.md §5.7) |
| **Buying journey** | Efficient, transactional, repeat-user behavior likely (may sell/buy multiple times) |
| **Content preferences** | Market-timing data, district-level trend content (content-system.md pillar 2) |
| **Visual preferences** | Data-forward, minimal decorative content |
| **Best CTA** | "List with confidence" / link to market report |

## 3. Relocating Seller

| Field | Detail |
|---|---|
| **Goals** | Sell quickly enough to align with a relocation timeline, without a distress-sale discount |
| **Fears** | Being forced into a lowball offer due to time pressure |
| **Pain points** | Juggling the sale alongside a move; limited time for the process |
| **Questions** | "Can this actually move fast without me losing value?" |
| **Objections** | "I don't have time to manage a complicated process right now" |
| **Emotional triggers** | Relief (a real, time-bound trigger — decision-triggers.md §1, not manufactured urgency) |
| **Buying journey** | Compressed timeline, values responsiveness and clear process above all else |
| **Content preferences** | Process-speed and clarity content, direct human-support messaging |
| **Visual preferences** | Simple, efficient, low-friction presentation |
| **Best CTA** | "Talk to a guide" |

## 4. Distressed/Urgent Seller

| Field | Detail |
|---|---|
| **Goals** | A fast, fair, respectful sale under real personal or financial pressure |
| **Fears** | Being taken advantage of because of visible urgency |
| **Pain points** | Emotionally sensitive context — requires the most careful, non-exploitative tone in the entire seller set |
| **Questions** | "Will I be treated fairly even though I need to move quickly?" |
| **Objections** | Reluctance to disclose the real reason for urgency |
| **Emotional triggers** | Relief, dignity, respect — never pressure-based messaging exploiting the situation (advertising-dna.md §5 forbids this explicitly) |
| **Buying journey** | Fast, private, minimal-friction preferred; often prefers direct human contact over public listing exposure |
| **Content preferences** | Discreet, respectful process messaging — not promotional content |
| **Visual preferences** | N/A — this persona is served through direct support contact, not broad campaign creative |
| **Best CTA** | "Talk to a guide" (private, human-first contact only — never a broad promotional CTA aimed at this persona) |

## 5. Inherited-Property Seller

| Field | Detail |
|---|---|
| **Goals** | Navigate an unfamiliar process (often involving multiple heirs) fairly and clearly |
| **Fears** | Family conflict over value/process; legal complexity around inherited property/deed transfer |
| **Pain points** | Multiple decision-makers (heirs) with potentially different priorities; unfamiliarity with the process since it wasn't planned |
| **Questions** | "What's the process for a property with multiple heirs?" "How do we agree on fair value among family?" |
| **Objections** | "This needs to go through a lawyer/family process first" |
| **Emotional triggers** | Confidence, Relief, respectful patience |
| **Buying journey** | Slower, multi-stakeholder, benefits heavily from clear, shareable, neutral information (market-psychology.md §6's family-influence pattern, applied to selling) |
| **Content preferences** | Process/legal-clarity educational content (plain-language per copywriting.md §5.4) |
| **Visual preferences** | Clear, neutral, document-friendly presentation |
| **Best CTA** | "Understand the process" |

## Best Practices
- Distressed and inherited-property sellers require the most careful tone calibration in the system — always route through direct human contact CTAs, never broad promotional messaging
- Seller content should validate pricing confidence without ever encouraging unrealistic expectations — trust (§1) depends on accuracy, not flattery

## Common Mistakes
- Treating all sellers as needing the same "sell fast" messaging — Investor-Sellers want speed, Distressed Sellers need dignity and privacy first
- Using urgency language toward relocating/distressed sellers in a way that could read as exploiting their real time pressure (psychological-triggers.md §4, §11 guardrails apply doubly here)
