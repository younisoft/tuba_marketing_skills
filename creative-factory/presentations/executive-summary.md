# Executive Summary — 8 Campaigns

> **Purpose:** the boardroom-readable cut. Each entry below is what you'd actually say out loud in a five-minute review, not the full working brief — full detail lives in `../campaigns/*.md`. Verdicts are the honest, as-scored results from each campaign's Brand Review, not rounded up.

---

### 01 — Brand Awareness
**The idea:** four real, specific vignettes prove "Everyone's Tuba is different" without ever explaining it — a keychain, a corner-shop smell, a pencil height-mark, a plain door. **Line:** "Everyone's Tuba is different." **Lead channel:** Snapchat (custom AR Lens) + YouTube brand film. **Verdict: PASS (Campaign 9.0 / Copy 9.1 / Design 8.8 / Brand 8.75).** Strongest creative-strength score in the set — the one device no competitor's name can copy.

### 02 — First Home Buyers
**The idea:** name the real fear ("what if something's wrong that I can't see") before the reader has to ask it, then answer it plainly. **Line:** "Understand the process." **Lead channel:** Instagram/TikTok educational carousel + video. **Verdict: PASS (Campaign 8.9 / Copy 9.0 / Design 8.7 / Brand 9.1).** Highest Audience Fit score in the set — built entirely from one named persona's real fears, not a generic first-time-buyer assumption.

### 03 — Property Investors
**The idea:** lead with one real, sourced number; let feeling arrive second. Anchors the recurring "دليل طوبة" monthly report as an ownable content property. **Line:** "See the numbers." **Lead channel:** LinkedIn. **Verdict: PASS (Campaign 8.6 / Copy 8.7 / Design 8.4 / Brand 8.9).** Honestly the least creatively distinctive campaign — data-first investment content is category-standard, not a Tuba invention; flagged rather than oversold.

### 04 — Real Estate Brokers
**The idea:** name the agent's real objection ("I've been burned by platforms that promised leads and delivered impressions") before pitching partnership. **Line:** "Grow with Tuba." **Lead channel:** LinkedIn. **Verdict: PASS (Campaign 8.6 / Copy 9.0 / Design 8.5 / Brand 9.0).** Expected Impact capped at 7 — the brief honestly admits it can't yet cite real lead-volume proof for a still-growing network.

### 05 — Developers
**The idea:** treat a developer's specific design choices with the same individualized respect an individual buyer's reasons get elsewhere in the system — brand-safety and phased structure as the actual proof, not a generic partnership pitch. **Line:** "Discuss your project." **Lead channel:** LinkedIn + partnership deck. **Verdict: PASS (Campaign 8.1 — the lowest Campaign average that still passes).** Acquisition-stage B2B work is inherently less creatively distinctive; stated honestly rather than inflated.

### 06 — Saudi National Day
**The idea:** refuse the category's single-image "we're proud too" convention entirely — a real, growing, user-submitted anthology of "this is my Saudi Arabia" stories, led by a Snapchat AR Lens. **Line:** "العز في أرضنا، والبيت في قلبها." **Lead channel:** Snapchat + Instagram. **Verdict: PASS (Campaign 8.7 / Copy 9.1 / Design 8.7 / Brand 8.9).** A genuinely award-plausible mechanic — a real UGC structure, not a produced hero image standing in for "the whole country."

### 07 — Ramadan
**The idea:** the room that matters most this month is never in the listing photos. Financing-term framing (no down payment until handover), never a price cut. **Line:** "لا دفعة أولى حتى الاستلام." **Lead channel:** Instagram + Snapchat. **Verdict: PASS (highest Copy average in the entire set — 9.3).** The pillar this system's emotional register was always going to serve best.

### 08 — Corporate Brand
**The idea:** institutional credibility content for regulators, media, and recruits — kept deliberately separate from consumer brand warmth. **Line:** "Every guide on our team has a different reason they chose this work." **Lead channel:** LinkedIn only (eight of thirteen formats explicitly excluded). **Verdict: REVISE (Campaign 7.6 — the only campaign in this set that does not cleanly PASS).** The narrative system has no document written for an institutional/regulatory audience — a real, structural gap this production run discovered, not a copy problem.
