# Creative Factory

> **Part of:** [`../AI_KNOWLEDGE_PLATFORM.md`](../AI_KNOWLEDGE_PLATFORM.md)
> **What this folder is:** the validation layer. Everything in `narrative-system/`, `advertising-system/`, and `brand-intelligence/` exists to *produce* real creative work — this folder is where that claim was tested, with 8 full campaigns built exactly as if Tuba's marketing team requested them this morning.
> **What this folder is not:** more documentation. Nothing here defines a new brand rule — every decision in every campaign traces to an existing document, cited inline. Where a real gap in the existing system was discovered during production, it's named honestly in [`../CREATIVE_FACTORY_REPORT.md`](../CREATIVE_FACTORY_REPORT.md), not quietly patched here.

## The One Rule This Entire Folder Follows

Every campaign was built by reading only what that specific task required (`../routing/TASK_ROUTER.md`'s discipline), never contradicting an existing rule, and never inventing new brand philosophy. Where the existing system had a genuine gap — most notably for Campaign 08 — the gap was reported, not silently filled in. That reporting discipline, not the campaigns themselves, is this folder's real deliverable.
