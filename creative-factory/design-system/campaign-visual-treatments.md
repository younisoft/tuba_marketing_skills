# Campaign Visual Treatments

> **Purpose:** the Art Director's cut across all 8 campaigns — how the same fixed brand system ([`../../advertising-system/visual-system.md`](../../advertising-system/visual-system.md)) flexes in content and emphasis without ever renegotiating a hex value, a type family, or the 60/30/10 color ratio (`advertising-dna.md §9`'s consistency rule, held without exception across every row below).
> **Not a second color system.** Every campaign below uses exactly Purple, Coral, and the same neutral scale. What changes is *which symbol, which icon pairing, and which register of restraint* each campaign leans on — the standing brand system's own built-in flexibility, not a new one invented per campaign.

## The One Rule Underneath All Eight Rows

Every symbol, icon pairing, and photography register above was *selected*, not invented — each traces to an existing document (`SYMBOL_LIBRARY.md`, `iconography.md §4`, `visual-system.md §6`). No campaign in this production run required a new visual asset class to be commissioned. Where a campaign (04, 05, 08) deliberately uses *no* narrative symbolism, that absence is itself a documented, reasoned choice — professional/institutional register correctly excludes the warmer symbolic language reserved for consumer-facing brand and seasonal work, per `copywriting.md §2.1`'s tone-by-context table.
